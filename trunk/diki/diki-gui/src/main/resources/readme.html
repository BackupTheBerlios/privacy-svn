Diki - the distributed tagging infrastructure

Getting started with diki!

Create bookmarks and share them with your friends! This system does not require a central server and only your friends are allowed to access your bookmarks!

  1. Go to the Menu -> configure -> Accounts. In the xmpp section, enter a username, password and choose any jabber server as communication server. Everything else is optional!
  2. done! click on Menu -> connect to see if it works. Hint: check the logger tab for more information.
  3. Go to the friends tab in the mainwindow and add friends you know. Hint: add this accounts:
    jtheuer@jabber.ccc.de
    the author
    dikibot@v-tml.uni-muenster.de
    It is a bot which already has a lot of bookmarks for you! Think of it as an example, once you have enough friends in your list you can remove him.
